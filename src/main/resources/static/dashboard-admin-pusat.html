Panduan Fungsi Dashboard
Kelola Gelombang (Periode)
Buat, edit, dan hapus periode/gelombang registrasi. Tentukan tanggal registrasi, ujian, dan pengumuman. Pilih jenis seleksi untuk setiap gelombang.
Kelola Jenis Seleksi
Atur jenis/jalur seleksi (Reguler, Beasiswa, dll). Tentukan harga, fasilitas, deskripsi, dan hubungkan dengan program studi yang tersedia.
Kelola Program Studi
Kelola daftar program studi yang tersedia. Tentukan kode, nama, tipe (kedokteran/non-kedokteran), dan status aktif/nonaktif program.
Kelola Akun
Kelola akun admin pusat, admin validasi, dan calon mahasiswa. Buat, edit, nonaktifkan akun dan atur role/hak akses pengguna.
Pesan Customer Service
Kelola pesan/komunikasi dari mahasiswa dan calon mahasiswa. Balas pertanyaan dan pantau percakapan customer service.
Pengumuman
Buat dan kelola pengumuman untuk seluruh sistem (hasil ujian, perubahan jadwal, info penting, dll).
Link Ujian
Kelola link Google Form ujian untuk setiap periode dan jenis seleksi. Sediakan akses ujian bagi mahasiswa.
Pengaturan Sistem
Atur konfigurasi sistem: link Google Form global, parameter validasi, dan setting lainnya.

Ekspor Data & Laporan

Info: Klik tombol "Buka Halaman" pada setiap kartu untuk membuka tabel data dengan fitur print, download CSV/JSON, dan pencarian lengkap.
Tabel Formulir Pendaftaran

Tabel lengkap semua formulir pendaftaran calon mahasiswa dengan informasi pribadi, asal sekolah, pilihan program studi, dan foto identitas.

Fitur: ✓ Tampilkan semua formulir | ✓ Download CSV/JSON | ✓ Print (Ctrl+P)
Tabel Daftar Ulang

Tabel lengkap data daftar ulang (registrasi ulang) mahasiswa dengan status validasi, catatan, tanggal submit, dan dokumen pendukung.

Fitur: ✓ Lihat semua reenrollment | ✓ Status validasi | ✓ Download & Print
Tabel Hasil Akhir (Detail)

Tabel lengkap hasil akhir dengan nomor registrasi, BRIVA, gelombang (wave type), tipe seleksi, dan program studi yang diterima.

Fitur: ✓ Info lengkap hasil akhir | ✓ Tipe gelombang & seleksi | ✓ Download & Print
Hasil Akhir (Filter Gelombang)

Tabel hasil akhir dengan filter berdasarkan gelombang (REGULAR_TEST, EARLY_NO_TEST). Download data spesifik per gelombang untuk keperluan admin.

Fitur: ✓ Filter by gelombang | ✓ Summary stats | ✓ Download filtered data
Semua Halaman: Dilengkapi dengan print (Ctrl+P), download CSV, dan download JSON format
